# BTC Trading Strategy Documentation

## Project Overview
Complete infrastructure for BTC/USDT trading strategy optimization.

## Modules
1. **Data Download** - Binance API integration
2. **Indicators** - Ichimoku Cloud calculations
3. **Backtesting** - Performance evaluation
4. **Risk Management** - Stop loss/take profit
5. **Strategies** - Trading logic implementations

## Usage
```bash
./setup.sh
source venv/bin/activate
python data/downloader.py
```

## Testing
Run unit tests:
```bash
python -m pytest tests/
```
